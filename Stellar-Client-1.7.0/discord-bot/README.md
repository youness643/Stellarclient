# Stellar Discord Bot 1.7.0

Bot di supporto per Stellar Client con slash command di stato, Store, saldo, quest, Premium e gestione Coins.

```powershell
npm install
npm run register
npm start
```

Il comando amministrativo per i Coins controlla il permesso Discord `Administrator` e il Guild ID configurato. Le modifiche passano dal backend mediante `STELLAR_API_KEY`.

Il bot token non è incluso. Configuralo con `configure-test.ps1` dopo averne generato uno nuovo.
